我强烈建议工程师都去申请使用 GitHub Copilot。相比传统的 chatbot 问答模式，Copilot 的 agent 能够直接帮你完成任务，比如分析日志、编写 RCA 等。  
另外，我还发现了一个很强大的工具叫 MCP，它可以让 Copilot 调用 Juniper 的各种工具和数据库，替你执行操作，比如部署实验环境、下发配置等。  
不过，如何让 AI 更高效地为自己所用，关键还是要靠自己不断实践，并总结经验。
エンジニアの皆さんには、ぜひGitHub Copilotの利用を強くおすすめしたいです。従来のチャットボットのようなQ&A形式と比べて、Copilotのエージェントはログ解析やRCAの作成など、タスクを直接実行してくれる点が大きな強みです。  
また、MCPという非常に強力なツールもあり、CopilotがJuniperの各種ツールやデータベースを連携して操作できるようになります。これにより、ラボ環境の構築やコンフィグの投入といった作業も自動化できます。  
ただし、AIをより効果的に活用するためには、実際に使いながら経験を積み、自分なりに工夫していくことが重要だと考えています。

越使用ai越迷茫，感觉完全在虚拟世界里工作形态，很快会被取代。ai取代不了的是与物理世界连接的工作，而这种工作需要的技能和思维，做了这么多年tac已经不知道是什么了。总之，眼前这2-3年内，人类社会形态会有很颠覆的改变，作为个体要怎样去应对这种变革是每个人都应该去思考的事情。

AIを使えば使うほど、完全に仮想世界の中で仕事をやってる人間はAIに代替されやすいと感じてます。
これから2〜3年のうちに、aiによって人間社会は大きく変わる可能性が高いと思います。その中で、個人としてどのようにこの変化に向き合い、どのようなにスキルを求められて、仕事だけではなくで、人生色々面で、真剣に考えなければならないと思います。

比如，与人打交道的能力、商业思维，以及当众表达和演讲的能力等。总的来说，在现实物理世界中的综合表现，将会变得越来越重要。
例えば、人と関わるコミュニケーション能力やビジネスの考え方、そして人前に話す・発表する能力などです。つまり、個人的な物理的な世界でのパフォーマンスが、これからもっともっt重要になっていくと思う。
