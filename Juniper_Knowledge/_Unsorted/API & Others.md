# RPC #
一种通信协议，让程序能像调用本地函数一样调用远程服务。
API 是抽象接口：定义“能做什么”（What），不关心“如何做”（How）。RPC 是具体实现：解决“如何远程调用”（How），属于 API 的一种实现方式。
gRPC（gRPC Remote Procedure Call）是一个高性能、开源的远程过程调用（RPC）框架，由 Google 开发并基于 HTTP/2 和 Protocol Buffers（Protobuf）构建。它主要用于微服务通信、分布式系统数据交换，并在云原生、网络设备（如 Juniper、Cisco）的Telemetry（遥测）系统中广泛应用。
