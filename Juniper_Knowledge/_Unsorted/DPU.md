# DPU #
是一种新型的可编程多核处理器，是一块...SoC...芯片，具有高算力和高性能网络接口。能高效计算处理数据并传到...CPU...和...GPU...上。
CPU...擅长通用计算，但...DPU...擅长基础层的应用任务，如网络协议处理，交换路由计算，加密解密，数据压缩这些脏活累活。

***SoC...芯片...***
system on chip ...数字芯片，是一种以确定系统功能为目标，各模块软硬件协同开发，最后把所有成果集中在一块芯片上的技术。里面装了...CPU,GPU...，内存控制器，通信模块等。
又被称为端侧芯片，在终端设备这一端，并承担了大部分的计算和处理任务。无需依赖远程服务器或云端来完成主要工作。被应用于很多终端设备中，如手机，...ipad...，电脑或者智能助手等...AI...设备等。
