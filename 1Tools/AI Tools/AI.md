# Github Org.
第一步：建 Organization（5分钟）
登录 github.com（ying-wang_hpeprod）
右上角头像 → Your organizations → New organization
选 Free 套餐
填 org 名字（比如 hpe-aft-team 或 junos-tools-team）
选 My personal account 作为 owner
第二步：建私有仓库

进入 org → New repository → 选 Private，比如：

copilot-skills — 存 skills、MCP 配置
case-tools — 存脚本工具
第三步：邀请团队成员

Org 页面 → People → Invite member → 输入每个人的 GitHub 用户名或邮箱

权限控制：

私有仓库默认外部人看不到
可以建 Teams（比如区分 maintainer/contributor）
每个人可以设置只读或可写
需要注意：不要把含 case 客户信息或 GNATS 查询结果的内容提交到 github.com，那类内容只能放 github.hpe.com。
